How to use the app

Idea of this project is to provide insight about users chess data. App works with lichess.org API, but any .pgn file should be suitable. Here is how you can use the app.

  1. Select the data set you want to inspect. You can begin with my data at lichess.org.The app uses Opening moves so it’s good idea to include them. Most of my games are Blitz games so it’s good idea to include the.
  2. When you have selected the data set copy API URL from the download page. Here are few samples
    • small dataset: approximately 1000 games and 1 minute to process. The link is place holder in url input field.
    • large dataset: approximately 5700 games and 7 minutes to process. It takes a while so I have included some screenshots below. If you want to process this dataset, please enter following link to url input field:
      • https://lichess.org/api/games/user/Haimis?tags=true&clocks=false&evals=false&opening=true&since=1356998400070&until=1636754400000&perfType=blitz
  3. Paste the link to the input field on this page and click “send link”. The app will download the dataset, parse it and provide some insights. Please note that this might take relatively long time with larger data sets. Unfortunately the app doesn’t provide any interim information about processing so it’s good idea to start exploring with a smaller data set.
